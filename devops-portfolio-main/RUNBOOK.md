## 📌 Project Overview
This project implements a complete CI/CD pipeline for a portfolio web application using modern DevOps tools. The goal is to automate build and deployment processes, reducing manual effort and ensuring faster delivery.

---

## 🎯 Problem Statement
Manual deployment of web applications is time-consuming, error-prone, and inefficient. There is a need for an automated pipeline to ensure reliable and fast deployment.

---

## 💡 Solution
A CI/CD pipeline is designed using GitHub Actions and deployed via Netlify. The system automatically builds and deploys the application whenever code is pushed to GitHub.

---

## 🛠️ Tools Used
- GitHub (Version Control)
- GitHub Actions (CI/CD)
- Docker (Containerization)
- Netlify (Deployment)

---

## 🔄 Architecture Diagram
Developer → GitHub → GitHub Actions → Docker → Netlify

---

## ⚙️ Workflow
1. Developer writes code
2. Code is pushed to GitHub
3. GitHub Actions triggers CI/CD pipeline
4. Application is built
5. Application is deployed to Netlify
6. Website becomes live
